🔑 Token-Helfer

Lege ein neues Zugangs-Token fest. Der Token selbst wird nirgends gespeichert — nur sein SHA-256-Hash kommt in config.js. Merke dir das Token; damit meldest du dich später an.


    
  

  

    So geht's:

    1. Token oben eintippen.

    2. Auf „Zeile kopieren" klicken.

    3. In config.js die Zeile accessHash: "…" durch die kopierte ersetzen, speichern.

    4. Committen & pushen → beim nächsten Öffnen gilt das neue Token.
